AnyLogic Cloud – веб-сервис для операционного использования моделей

AnyLogic Cloud — это защищённая веб-платформа для запуска имитационных моделей. Проводите эксперименты в облаке, работайте над моделью в команде и обменивайтесь результатами онлайн. AnyLogic Cloud упрощает операционное использование имитационных моделей.

Попробовать AnyLogic CloudВерсии AnyLogic Cloud

Используйте модель для решения операционных задач

Превратите имитационную модель в платформу для поддержки принятия оперативных решений. Храните модели в облаке и интегрируйте их с оперативными данными, проводите эксперименты и делитесь результатами с конечными пользователями, чтобы они могли сразу применять результаты моделирования для решения текущих задач. Модель в AnyLogic Cloud проще интегрировать в рабочие процессы и использовать для создания цифровых двойников.

Представляйте модели клиентам онлайн

Используйте готовую облачную среду, чтобы делиться моделями с клиентами: для их запуска не нужно устанавливать дополнительное ПО. В AnyLogic Cloud можно управлять доступом к моделям, проще обновлять их и создавать собственные интерфейсы для моделей.

Коллективный доступ к модели

Используйте AnyLogic Cloud для работы над моделью в команде. Управляйте правами доступа и отправляйте ссылки на модели коллегам или клиентам, чтобы они могли запускать их удалённо. Пользователи смогут настроить эксперименты с помощью интерактивных панелей управления, посмотреть анимацию и получить результаты проведённых экспериментов из общей базы данных.

Продвинутое управление сценариями

Храните, систематизируйте и редактируйте данные экспериментов и сценариев в облаке, включая версии моделей, параметры запусков и результаты моделирования. Настраивайте и запускайте различные эксперименты, в том числе варьирование параметров и Монте-Карло. Сервис также позволяет визуально сравнивать результаты экспериментов, в том числе разницу ключевых показателей в разных прогонах.

Параллельные вычисления для сложных экспериментов

В AnyLogic Cloud многопрогонные эксперименты выполняются быстрее и эффективнее, чем на обычном компьютере. Сервис оперативно реагирует на потребности в вычислительных мощностях, подключая дополнительные узлы и ядра для выполнения экспериментов.

Все эксперименты вместе со входными данными и результатами хранятся в облаке. Если входные данные у запусков одинаковы, AnyLogic Cloud использует предыдущие результаты, ускоряя новый эксперимент и экономя вычислительные ресурсы.

Панели управления для расширенной аналитики

Для визуализации результатов экспериментов в AnyLogic Cloud есть масса графиков и диаграмм, включая продвинутые – например, диаграмма размаха, точечная диаграмма и диаграмма поверхности. Используйте их и другие элементы, чтобы создать собственную панель для управления моделью и анализа результатов.

Создавайте онлайн-решения на основе имитационных моделей

AnyLogic Cloud – это готовая среда для разработки продуктов на основе имитационных моделей. Вы можете создать собственный облачный сервис с пользовательским интерфейсом, визуализацией и анимацией, подключив его по Cloud API к модели в облаке.


Пример готового решения

Ускоряйте машинное обучение

В AnyLogic можно создавать динамические модели для обучения и тестирования решений на основе ИИ. Загрузите модель в AnyLogic Cloud и подключите её к платформе машинного обучения с помощью RESTful API на Python, Java и JS. Модели в AnyLogic Cloud используют мощные облачные вычисления, которые позволяют нейронным сетям обучаться быстрее и эффективнее.


AnyLogic и ИИ

AnyLogic Cloud – публичный сервис для запуска моделей. Его также можно установить на частной ИТ-инфраструктуре компании. Выберите версию, которая подходит для ваших бизнес-задач.

Публичная версия AnyLogic Cloud доступна для всех. Даже если вы не пользуетесь ПО AnyLogic, вы можете запускать модели в веб-сервисе. Все пользователи AnyLogic могут загружать модели в Cloud и делиться ими, а те, у кого есть действующая лицензия AnyLogic с техподдержкой, могут также ограничивать доступ к облачным моделям.

При использовании подписки модель работает в AnyLogic Cloud неограниченное время, а ресурсоёмкие эксперименты в облаке выполняются быстрее. Подписка также позволяет обращаться из модели к сторонним ресурсам — например, базам данных — и пользоваться API AnyLogic Cloud для изменения интерфейса.

Private Cloud можно развернуть на собственной ИТ-инфраструктуре или на мощностях PaaS-провайдера. Таким образом компаниям будет проще настроить веб-сервис в соответствии с корпоративными требованиями безопасности, контролировать конфиденциальные данные и интегрировать имитационные модели в системы обработки данных.

Private Cloud доступен в трех версиях:

  • Private Cloud Enterprise можно установить на несколько серверов для повышения производительности и неограниченного количества параллельных запусков моделей.
  • Private Cloud Lite и Private Cloud Pro – это полнофункциональные и доступные версии Private Cloud Enterprise. Они устанавливаются как классические программы на сервер в локальной сети. Для Private Cloud Lite максимальная производительность сервера ограничена 16 ядрами, для Private Cloud Pro это число увеличено до 64.
